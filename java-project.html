Project Overview

This project demonstrates advanced Object-Oriented Programming concepts. The application was designed to read raw data from external files, process the information using custom sorting and searching algorithms, and generate formatted reports.

Key Challenges Solved

  • Implemented error handling to manage corrupt or missing data files.
  • utilized encapsulation to protect data integrity.
  • Optimized data parsing logic to handle multiple file formats.
